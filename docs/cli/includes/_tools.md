Display the status of Quarto installed dependencies

``` {.bash}
quarto tools 
```


## Options

|Flags          |Arguments |Description     |
|:--------------|:---------|:---------------|
|`-h`, `--help` |          |Show this help. |


## Commands

|Command |Description                                  |
|:-------|:--------------------------------------------|
|`help`  |Show this help or the help of a sub-command. |


## Examples
### Show tool status

``` {.bash filename='Terminal'}
quarto tools
```
